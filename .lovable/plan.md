

# CardPerks — Premium Rehaul Plan

## Overview
Transform the CardPerks website into a premium, immersive credit card perks and voucher tracking platform with a dark theme, gold accents, 3D animations, and polished navigation. Starting with the **Homepage + Navigation** using **static mock data** and **UI-only auth pages**.

---

## 1. Global Navigation Bar
- Sticky top nav with the CardPerks logo, links: **Home, Vouchers, Know Your Cards, Perk AI, Guides Hub, More** (dropdown)
- Right-side icons: favorites (heart), notifications, dark mode toggle, user avatar
- Gold accent on active link with subtle underline animation
- Mobile: hamburger menu with slide-in drawer
- Smooth scroll-aware: transparent on hero, solid on scroll

## 2. Homepage — Hero Section
- **3D animated background** using Three.js: floating geometric shapes (cards, coins, abstract polygons) with slow rotation and parallax depth
- Large serif heading: "Track Voucher Rates. Maximize Savings." with a gold gradient on "Maximize Savings"
- Tagline text with fade-in animation
- Two CTAs: "Browse Vouchers →" (solid gold) and "Compare Cards" (outlined)
- Stats bar: "519 Brands · 160 Cards · 5 Platforms · Updated Daily" with animated count-up numbers

## 3. Homepage — How It Works Section
- Three feature cards: Track Voucher Rates, Compare Cards, Savings Dashboard
- Each card has an icon, title, description, and subtle tilt-on-hover 3D effect
- Staggered fade-in animation as user scrolls into view

## 4. Homepage — Popular Vouchers Section
- Horizontal scrollable card carousel showing top brand vouchers (Flipkart, Zomato, Uber, BigBasket, Amazon)
- Each card shows: brand logo, name, category tag, discount badge, brief description
- Hover effect: slight 3D lift with shadow expansion
- "View All" link to Vouchers page

## 5. Homepage — Explore More Section
- Three cards linking to: Know Your Cards, Perk AI, Guides Hub
- Each with icon, title, description, and CTA button
- Subtle scale-on-hover animation

## 6. Homepage — Footer
- Brand info, copyright "©2026 CardPerks"
- Links: About Us, Privacy Policy
- Newsletter email signup input (UI only)
- Clean, minimal dark footer with gold accent dividers

## 7. 3D Credit Card Component
- Interactive 3D credit card model that can rotate and tilt on mouse movement
- Used on the hero section as a floating decorative element
- Realistic card design with reflective/metallic material effect
- Smooth idle animation when not being interacted with

## 8. Auth Pages (UI Mockup Only)
- **Sign Up page**: CardPerks branding, name/email/password form, feature bullets (Save favourites, Track expenses, AI recommendations)
- **Login page**: Email/password form, "Forgot password?", "Remember me", social login buttons (Google), "Sign Up" redirect
- Premium glass-morphism card design on a dark background
- No real authentication — just the visual pages

## 9. Design System & Polish
- Dark theme as default with gold (#C4A35A) accent color throughout
- Serif font for headings, clean sans-serif for body
- Smooth page transitions and scroll-triggered animations
- Consistent card styles with glassmorphism/frosted glass effects
- Responsive design across desktop, tablet, and mobile
